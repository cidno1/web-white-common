# Notes for study

### 정수 변환 함수들
- `Math.floor()`: 매개변수를 가장 가까운 정수로 내립니다.

### 배열 메소드
- `Array.push()` : 매개변수를 해당 배열에 추가한다. 리턴값은 추가한 값.
- `Array.join()` : 매개변수로 전달한 값을 배열요소들 사이에 들어가 연결된다. 연결한 배열요소들을 문자열로 반환한다.
                 해당 배열을 수정하지 않는다.
